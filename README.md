# FINZ – Financial Management Web Application

## 📖 Overview

FINZ is a full-stack web-based financial management system designed to help users manage income, expenses, and financial organization in a structured and intuitive way.

The system was independently designed, developed, and deployed as a production-level application. It demonstrates practical implementation of modern web technologies, system architecture design, and real-world deployment.

🔗 **Live System:** https://finz.net.br  
🔗 **Repository:** https://github.com/Fjailton16/FINZ-Main

- Hosted on Hostinger

- Version control using Git  

---

## 🧱 System Architecture

FINZ follows a modern component-based web architecture:

- **Frontend Layer:** React-based SPA with modular components  
- **State Management:** Structured client-side state handling  
- **Backend Integration:** RESTful interaction via Supabase services  
- **Database:** Cloud-hosted relational database (PostgreSQL)  
- **Authentication:** Secure user authentication and session handling  

The architecture was designed to ensure scalability, modularity, maintainability, and performance.
